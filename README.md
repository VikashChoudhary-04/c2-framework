# Educational C2 Framework (Lab Only)

This repository contains a learning-focused Command-and-Control (C2)
framework built to understand:

- Secure agent/server communication
- Encrypted tasking
- OPSEC fundamentals
- Controlled lab testing

⚠️ This project must only be used in isolated, authorized environments.

## Components
- Agent (Linux Python implant)
- Flask-based HTTPS C2 server
- AES-GCM encrypted communication
